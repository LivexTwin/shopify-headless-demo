# MALYXIA

A modern fashion label focused on minimal essentials and refined visual systems.

## Overview

MALYXIA is a design-led clothing brand built around clarity, structure, and restraint. Each piece is developed with attention to form, fabric, and presentation.

This repository contains the official site and product system powering the MALYXIA storefront.

## Tech

- Astro
- Svelte
- Shopify Storefront API
- TypeScript

## System

- Static-first architecture for speed and reliability
- Component-based product rendering
- Paired product imagery (front / back consistency system)
- Lightweight client interactions for navigation and cart

## Product Structure

Each product follows a fixed visual system:

1. Front image
2. Back image
3. Optional detail or variant imagery

This ensures consistency across the entire catalog.

## Design Principles

- Minimal UI
- Fast load performance
- Image-first experience
- No unnecessary visual noise
- Consistent product presentation

## Deployment

The site is deployed as a static build and can be hosted on any modern edge platform.

## Notes

This system is intentionally kept lightweight. The focus is on product clarity, not interface complexity.
